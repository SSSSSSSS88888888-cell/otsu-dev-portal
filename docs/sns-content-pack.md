## 3. TikTok台本テンプレート（3種）

### テンプレA: 15秒「一発デモ型」
課題提示→デモ→オチの三段構造。

```
[0-2秒] テロップ大：「これ、ヤバくないですか」
        画面：問題提起のスクショ（例: 長すぎる会議）

[2-7秒] 画面：アプリ操作の手元動画（縦画面全面）
        声：「で、作ったのがこれです」

[7-13秒] 画面：結果画面の大きな数字
         声：「見てください、この数字」
         テロップ：「¥10,000 ← この会議の価値」

[13-15秒] 声：「やる気、出ません？」
          テロップ：「@otsu_dev / 100本チャレンジ」
```

### テンプレB: 30秒「ストーリー型」
きっかけ→作った→使い方→結果感想。

```
[0-3秒] 声：「残業代、秒で積み上がるの見たくないですか」
        画面：定時後のオフィスの時計

[3-8秒] 声：「というわけで、作りました」
        画面：アプリを開く瞬間

[8-18秒] 画面：時給入力→残業開始→数字が積み上がる
         声：低テンションで実況
         テロップで金額を強調

[18-25秒] 声：「1時間で¥1,875、2時間で¥3,750」
          画面：貯金箱に変わっていく演出

[25-30秒] 声：「残業、ちょっとマシになりました」
          テロップ：「@otsu_dev / 毎日1本作ってます」
```

### テンプレC: リアクション型（可変尺）
ユーザーのDMやツイートを画面に表示→開発者として反応。

```
[0-3秒] 画面：「こんなDMいただきました」テロップ＋DM画像

[3-10秒] 画面：DMを読み上げ
         声：低テンションで「なるほど」

[10-20秒] 画面：アプリの該当機能にフォーカス
          声：「こういうのが役に立ってるなら、作った甲斐がありました」

[20-30秒] 声：「明日もまた1本、作ります」
          テロップ：「@otsu_dev / 100本チャレンジ」
```

### TikTok共通の必須要素
- 縦9:16で撮影
- テロップ全編必須（ミュート視聴対応）
- 最初の1.5秒で「え、何これ」を演出
- BGMはトレンド音源から選ぶ
- 最後にフォロー誘導テロップ
